# Example: Join & Quit Messages

## OVERVIEW

Instead of having the default join and quit messages, you can create fancy, custom ones using Skript.

## CODE

```text
on join:
  set join message to "[+] %player%"

on quit:
  set quit message to "[-] %player%"
```
